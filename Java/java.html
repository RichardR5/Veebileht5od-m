Java - rakenduste selgroog

  • Keerukus: keskmine
  • Populaarsed raamistikud: Spring, Hibernate, Strut
  • Kasutus: Veebi-, Android- ja pilverakendused, Big Data, manussüsteemid
  • Populaarsus: #2 PYPL Popularsusindeksi järgi (Märts 2021), #2 Tiobe Indeksi järgi (Märts 2021), 44.1% StackExchange'i arendajatest soovitab (2020).

Java on jäänud de-facto programmeerimiskeeleks ettevõtte-tasemel rakenduste loomisel ja arendamisel juba rohkem kui 20 aastat.

Sun Microsystems'i James Goslingi poolt 1995. aastal loodud objektorienteeritud programmeerimiskeel Java on sellest ajast alates olnud arendajatele turvaline, usaldusväärne ja skaleeritav tööriist. Mõned Java pakutavad funktsioonid, mis muudavad selle eelistatumaks kui mitmed teised programmeerimiskeeled, on selle "prügikoristusvõimalused", tagasiühilduvus, platvormi sõltumatus JVM-i kaudu, teisaldatavus ja suur jõudlus. Java populaarsust võib selgelt näha Fortune 500 liikmete seas, sest 90% neist kasutab Javat oma äritegevuse tõhusaks juhtimiseks.

Mille jaoks seda keelt kasutatakse - peale selle, et seda kasutatakse töökindlate ärirakenduste arendamiseks, on Java ka Android-platvormil laialdaselt kasutatud, muutes selle eelduseks Androidi arendajale. Java võimaldab arendajatel luua rakendusi mitmesuguste tööstusharude jaoks nagu pangandus, elektrooniline kauplemine, e-kaubandus, rakendused hajusarvutuseks. Samuti arendatakse Javaga chatbote, pilveteenuseid ning pisut ka IoT (internet of things) seadmeid. Javat kasutavad näiteks Airbnb, Pinterest, Uber, Spotify, Slack, Amazon, Google ja teised.
